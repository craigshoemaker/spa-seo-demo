What is Hijax?

Hijax is the progressive enhancement concept of intercepting link clicks via JavaScript.

The links (anchors) include normal href values that work just as well for browsers that have JavaScript disabled as they do for search engine bots. When JavaScript is present (in a normal web browser) the click action on the link is intercepted or "hijacked" and an Ajax request is initiated rather than allow the standard full-browser request for the page.

The following code listing demonstrates how you can select all relative links in jQuery: 

var selector = '#content-container a[href^="/"], ' +
               '#content-container a[href^="' + window.location.origin + '"]';
            
$(selector).on('click', function (e) {
    e.preventDefault();
    var href = this.getAttribute('href');
    // do some processing here
    return false;
});

The Hijax approach is often used in conjunction with either the hash/hash-bang or history implementations to keep the URL updated during Ajax requests. Read more on the architectural overview.
