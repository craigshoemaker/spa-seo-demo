What is HTML History?

Use of the HTML History API allows you to make changes to the browser's location without initiating an explicit request back to the server.

This is valuable because you can change the location through code and maintain explicit control over how your application behaves. for instance you may choose to manually update the browsers location and then request partial page content via an Ajax request. the benefit to this approach is that as you keeping the location updated, should a request be made to that specific location in your application you have the opportunity to render a snapshot of the HTML that corresponds to the state of the application based on the given URL.
